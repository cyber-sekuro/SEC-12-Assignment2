# TaskMaster Pro API

A RESTful API built with **Node.js, Express, Sequelize, and PostgreSQL** for managing users and their tasks.

This project allows users to create accounts and manage tasks assigned to them.

---

# Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Sequelize ORM
- dotenv
